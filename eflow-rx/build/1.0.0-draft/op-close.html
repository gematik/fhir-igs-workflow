Task schliessen ($close)

Nachricht

Die Nachricht wird als HTTP POST an /Task/{id}/$close gesendet.

Anforderungen an Schnittstelle

  • FD-Anforderungen zu $close: Anforderungen an den E-Rezept-Fachdienst zur Bereitstellung der Schnittstelle.
  • Client-Anforderungen zu $close: Anforderungen an den Client des E-Rezept-Fachdienstes zur Nutzung der Schnittstelle.
