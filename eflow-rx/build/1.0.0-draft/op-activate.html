Task aktivieren ($activate)

Diese Operation basiert auf der gleichnamigen Operation in der Core-Spezifikation.

Nachricht

Die Nachricht wird als HTTP POST an /Task/{id}/$activate gesendet.

Anforderungen an Schnittstelle

  • FD-Anforderungen zu $activate: Anforderungen an den E-Rezept-Fachdienst zur Bereitstellung der Schnittstelle.
  • Client-Anforderungen zu $activate: Anforderungen an den Client des E-Rezept-Fachdienstes zur Nutzung der Schnittstelle.
