Task akzeptieren ($accept)

Nachricht

Die Nachricht wird als HTTP POST an /Task/{id}/$accept gesendet.

Anforderungen an Schnittstelle

  • FD-Anforderungen zu $accept: Anforderungen an den E-Rezept-Fachdienst zur Bereitstellung der Schnittstelle.
  • Client-Anforderungen zu $accept: Anforderungen an den Client des E-Rezept-Fachdienstes zur Nutzung der Schnittstelle.
