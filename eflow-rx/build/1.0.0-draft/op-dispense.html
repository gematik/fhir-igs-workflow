Dispensierinformationen bereitstellen ($dispense)

Nachricht

Die Nachricht wird als HTTP POST an /Task/{id}/$dispense gesendet.

Anforderungen an Schnittstelle

  • FD-Anforderungen zu $dispense: Anforderungen an den E-Rezept-Fachdienst zur Bereitstellung der Schnittstelle.
  • Client-Anforderungen zu $dispense: Anforderungen an den Client des E-Rezept-Fachdienstes zur Nutzung der Schnittstelle.
