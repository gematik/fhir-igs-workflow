Query API: Task

Diese Seite beschreibt den Einstieg in die Task-Query-Schnittstellen des RX-Moduls.

Nachricht

Die Task-Query wird ueber HTTP GET auf /Task und /Task/{id} aufgerufen.

Anforderungen an Schnittstelle

  • FD-Anforderungen zur Task-Query: Anforderungen an den E-Rezept-Fachdienst zur Bereitstellung der Schnittstelle.
  • Client-Anforderungen zur Task-Query: Anforderungen an den Client des E-Rezept-Fachdienstes zur Nutzung der Schnittstelle.
