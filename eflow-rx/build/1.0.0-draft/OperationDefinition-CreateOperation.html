OperationDefinition: CreateOperation

Official URL: https://gematik.de/fhir/erp/OperationDefinition/CreateOperation Version: 1.0.0-draft
Draft as of 2025-12-15 Computable Name: Create

Diese Operation erstellt eine Aufgabe für die Verschreibung eines bestimmten Flowtypes.

URL: [base]/Task/$create

Input parameters Profile:Parameters

Output parameters Profile:GEM ERP PR CreateOperation Output

Parameters

UseNameScopeCardinalityTypeBindingDocumentation
INworkflowType1..1CodingValueSet der FlowType-Codes (Required)

Dieser Parameter fordert die Erstellung eines Tasks mit bestimmten Flowtype an.

OUTreturn1..1Task

Der erstellte Task mit einer flowType-spezifischen Rezept-ID
