Task zurueckweisen ($reject)

Nachricht

Die Nachricht wird als HTTP POST an /Task/{id}/$reject gesendet.

Anforderungen an Schnittstelle

  • FD-Anforderungen zu $reject: Anforderungen an den E-Rezept-Fachdienst zur Bereitstellung der Schnittstelle.
  • Client-Anforderungen zu $reject: Anforderungen an den Client des E-Rezept-Fachdienstes zur Nutzung der Schnittstelle.
