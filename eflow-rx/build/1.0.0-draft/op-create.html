Task erzeugen ($create)

Diese Operation basiert auf der gleichnamigen Operation in der Core-Spezifikation.

Nachricht

Die Nachricht wird als HTTP POST an /Task/$create gesendet.

Anforderungen an Schnittstelle

  • FD-Anforderungen zu $create: Anforderungen an den E-Rezept-Fachdienst zur Bereitstellung der Schnittstelle.
  • Client-Anforderungen zu $create: Anforderungen an den Client des E-Rezept-Fachdienstes zur Nutzung der Schnittstelle.
