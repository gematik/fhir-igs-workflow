Query API: AuditEvent

Diese Seite beschreibt die moduluebergreifende AuditEvent-Query-Schnittstelle.

Nachricht

Die AuditEvent-Query wird ueber HTTP GET auf /AuditEvent und /AuditEvent/{id} aufgerufen.

Anforderungen an Schnittstelle

  • Server-Anforderungen zur AuditEvent-Query: Anforderungen an den E-Rezept-Fachdienst zur Bereitstellung der Schnittstelle.
  • Client-Anforderungen zur AuditEvent-Query: Anforderungen an den Client des E-Rezept-Fachdienstes zur Nutzung der Schnittstelle.
