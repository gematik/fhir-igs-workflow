Operation API: Zugriffsberechtigung einsehen

Seiteninhalt:

  • Nachricht
  • Anforderungen an Schnittstelle
  • API Beschreibung

Mit der FHIR-Operation kann die Zugriffsberechtigung für die im ACCESS_TOKEN angegebene KVNR abgerufen werden. Diese Operation steht Versicherten zur Verfügung.

Nachricht

Die Nachricht wird als HTTP GET an /$read-eu-access-permission gesendet.

Anforderungen an Schnittstelle

  • Server Anforderungen zu $read-eu-access-permission: Anforderungen an den TI-Flow-Fachdienst zur Bereitstellung der Schnittstelle.
  • FdV-Anforderungen zu $read-eu-access-permission: Anforderungen an das E-Rezept-FdV zur Nutzung der Schnittstelle.
