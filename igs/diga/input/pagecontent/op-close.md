Operation: $close

Frage: Was ist mit den folgenden Afos?
A_19230-01 - E-Rezept-Fachdienst - Task schließen - Rollenprüfung
A_26003-01 - E-Rezept-Fachdienst - Task schließen - Flowtype 162 - Profilprüfung MedicationDispense


## Nachricht
Die Operation $close wird als HTTP POST auf /Task/<id>/$close ausgeführt.

## Auslöser
Der Kostenträger übermittelt die Abgabeinformation (Freischaltcode) und
schliesst den Vorgang ab.

## Hinweis
Es gelten die Vorgaben des Basis-Workflows für E-Rezepte.
