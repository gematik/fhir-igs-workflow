Query API: MedicationDispense

MedicationDispense wird fuer die Abgabeinformation der DiGA genutzt. Der
Freischaltcode sowie die DiGA-Bezeichnung werden in der Abgabeinformation
bereitgestellt. Falls kein Freischaltcode bereitgestellt werden kann, enthaelt
die Abgabeinformation eine begruendende Rueckmeldung.

## GET /MedicationDispense
- Abruf der Abgabeinformationen fuer den Versicherten
